**Idea Web**: The Creative Capture Engine for OdinPad****



---

**Purpose & Vision**

**The Idea Web is OdinPad's lightweight, always-accessible "idea logging" tool—designed to defeat blank-page syndrome at the source. It serves as the entry point for every story spark, random thought, fragment, or "what if" moment, ensuring nothing gets lost in notes apps, scraps of paper, or forgotten voice memos.**

**Unlike a generic notes app, the Idea Web is deeply integrated with OdinPad's ecosystem: ideas auto-link to the Story Wiki, Story Canvas, Manuscript, and gamification system. It encourages daily micro-habits while remaining optional and non-intrusive—perfect for pantsers who hate rigid structure early on.**

**Core Philosophy: Capture first, organize second, structure when ready.**

---

**Core Philosophy: Ideas First, Structure Later**

**Idea Web operates on a simple principle: capture first, organize second, structure when ready.**

**Traditional writing tools often demand too much structure too early. Opening a new project file feels like a commitment. Idea Web removes that psychological barrier by providing:**

- **Instant capture — one-click new ideas, no project setup required**
- **Visual exploration — see how ideas connect before deciding what matters**
- **Gentle organization — tags, links, and visual clusters without mandatory hierarchies**
- **Low-pressure environment — ideas live here until they're ready to become something more**
- **Rewarding micro-habits — logging one idea earns points and protects daily streaks**

**The tool serves as both a personal creative sketchbook and a powerful connection engine, helping writers discover the threads that tie their disparate ideas into something greater.**

**How It Fits OdinPad's Mission:**

- **Addresses blank-page syndrome by making idea capture frictionless and rewarding**
- **Prevents lost fragments that later cause inconsistency or plot holes**
- **Bridges ideation → planning → drafting without tool-switching**
- **Aligns with AI philosophy (0% generative—editorial only for detection, suggestions, and connections)**
- **Embodies the cozy minimalist brand—a warm, inviting space where creativity begins**

**Core Features**

**1. Instant Capture**

**The heart of Idea Web is frictionless entry. Every idea should take seconds to record, not minutes.**

**Capture Modes:**


|                          |                                                                                                                                                                                                                                                                    |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Mode**                 | **Description**                                                                                                                                                                                                                                                    |
| **Quick Entry Modal**    | **Keyboard shortcut (Cmd/Ctrl + Shift + I or floating "+" button) opens minimalist popover anywhere in OdinPad. Type title, optional description, save. Auto-saves with timestamp and optional mood/tag (e.g., "euphoric plot twist," "dark antagonist motive").** |
| **Daily Seed Prompt**    | **Gentle, opt-in daily nudge in dashboard (e.g., "What's one image/scene/feeling in your head today?"). Ties into habit streaks and rewards consistency.**                                                                                                         |
| **Mobile Quick Capture** | **PWA-optimized for phone capture—works during walks, commutes, or middle-of-the-night inspiration. Voice-to-text support. Syncs instantly to desktop.**                                                                                                           |
| **Email-to-Idea**        | **Forward snippets, articles, or voice transcriptions to a personal Idea Web email address. They appear as new ideas automatically with source preserved.**                                                                                                        |
| **Browser Extension**    | **Highlight text on any webpage, click extension, and save as an idea with source URL and page title attached. Perfect for research capture.**                                                                                                                     |
| **Drag & Drop Import**   | **Drag images, PDFs, text snippets, or screenshots directly into Idea Web—content is saved as an idea with media attachment.**                                                                                                                                     |


**Idea Fields (Minimum Required):**

- **Title (required—just a few words to trigger memory)**
- **Description / snippet (optional—can be empty or just a sentence)**
- **Source type (original, inspired by, research, quote, dream, overheard, etc.)**
- **Tags (optional—auto-suggest existing)**
- **Mood / emotional tone (optional—for later thematic clustering)**
- **Related ideas (optional linking as you capture)**

**2. Flexible Organization**

**Idea Web rejects rigid hierarchies in favor of organic, multi-dimensional organization.**

**Tagging System:**

- **Unlimited tags per idea**
- **Tag hierarchies (e.g., Fantasy > Magic Systems > Hard Magic)**
- **Color-coded tag groups for visual scanning**
- **Tag combinations as saved filters (e.g., "show all seeds tagged dark fantasy + protagonist")**
- **Auto-suggest existing tags while typing—prevents tag sprawl**

**Idea Statuses:**


|               |                                                     |                                            |
| ------------- | --------------------------------------------------- | ------------------------------------------ |
| **Status**    | **Meaning**                                         | **Visual Cue**                             |
| **Seed**      | **Just captured, no development yet**               | **Sprout icon, muted color**               |
| **Sprouting** | **Active exploration, added notes, revisiting**     | **Growing plant icon, warm accent**        |
| **Growing**   | **Promising, regular attention, gaining detail**    | **Vibrant color, leaf icon**               |
| **Dormant**   | **Set aside for later—not abandoned, just resting** | **Grayed out, snowflake icon**             |
| **Harvested** | **Moved to a full project**                         | **Harvest basket icon, linked to project** |
| **Archived**  | **No longer pursuing (but never deleted)**          | **Box icon, hidden from main views**       |


**Idea Types (Optional Categories):**

- **Character concept**
- **Plot fragment / scene idea**
- **Setting / location**
- **Magic / technology system**
- **Dialogue snippet (with optional speaker attribution)**
- **Theme / philosophical idea**
- **Scene image / visual inspiration**
- **Research note (with source link)**
- **Dream / subconscious material**
- **Writing prompt (self-generated or discovered)**
- **Title / name idea**
- **Conflict / obstacle concept**

**3. Visual Idea Mapping**

**The signature feature of Idea Web: seeing connections visually.**

**Mind Map View:**

- **Central node = selected idea, tag group, or project**
- **Connected nodes = linked ideas, tags, or related concepts**
- **Drag to reorganize layout (positions saved per user)**
- **Double-click any node to open idea detail drawer**
- **Color-coded by idea type, status, mood, or custom groups**
- **Zoom and pan for large maps (supports hundreds of nodes)**
- **Export map as image (PNG/SVG) or interactive HTML for sharing**

**Connection Suggestions (AI Editorial, Non-Generative):**

**AI analyzes idea content and suggests potential connections based on:**

- **Shared tags and overlapping tag hierarchies**
- **Similar language patterns (keyword matching, thematic resonance)**
- **Complementary idea types (e.g., a character + a setting that fit together)**
- **Time proximity (ideas logged near each other in creative bursts)**
- **Cross-references to Story Wiki entries**

***User controls:* Approve, dismiss, or mark "not relevant." Connection intelligence improves over time based on user feedback.**

**Cluster Detection:**

- **Automatic detection of idea clusters (5+ ideas with strong connections to each other)**
- **Cluster highlights appear in map view with halo effect**
- **Click cluster to see all member ideas**
- **Option to convert a cluster into a project template or outline starter**

**Timeline View Alternative:**

- **For linear thinkers: sort ideas chronologically with connection lines visible between related entries**
- **Filter by date ranges to see creative evolution over time**

**6. Project Integration**

**Idea Web is not a silo—it's the gateway to OdinPad's full writing environment.**

**Idea → Project Workflow ("Harvesting"):**

1. **Select one or more ideas (checkbox selection in list view or map view)**
2. **Click "Harvest" button**
3. **Choose destination:**
  - **New Project → transforms selected ideas into a new project with optional template (novel, short story, series, screenplay, etc.)**
  - **Existing Project → add ideas to current project's relevant sections**
4. **Ideas transform based on type:**
  - **Character ideas → Character profiles in Story Wiki**
  - **Setting/location ideas → Location entries in Story Wiki**
  - **Plot fragments → Story Canvas cards (drag to arrange)**
  - **Dialogue snippets → Manuscript (insert at cursor position)**
  - **Research notes → Research Hub with source preservation**
  - **Magic/tech systems → System entries in Story Wiki**
5. **Original ideas remain in Idea Web with "Harvested" status and link to new project**
6. **Harvest confirmation shows what was created and offers to open new project**

**Two-Way Sync (Configurable):**

- **Changes to harvested content in main project can optionally reflect back to Idea Web**
- **Keeps the idea seed visible as the story grows**
- **Settings per idea: "sync updates" or "freeze original"**
- **Useful for tracking how an idea evolved during drafting**

**Project Ideas Tab:**

- **Each OdinPad project has an "Idea Web" tab showing related seeds and harvested ideas**
- **Quick capture from within project context (shortcut works anywhere)**
- **Project-specific tags auto-apply to ideas captured while project is active**

**Idea Prompts (AI Editorial, Non-Generative):**


|                          |                                                                                                                                                                                                                                    |
| ------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Tool**                 | **Description**                                                                                                                                                                                                                    |
| **"What If?" Generator** | **Suggests alternative directions based on existing idea. Example: "You have a wizard who lost their memory. What if they were the villain who erased their own memories?" User writes the exploration—AI only provides prompts.** |
| **Randomizer**           | **Pulls one existing character idea + one setting idea + one conflict idea from your Idea Web and presents them as a combination prompt. User decides if it sparks something.**                                                    |
| **Combine Two Ideas**    | **Select two ideas, click "Combine"—AI suggests ways they might merge (e.g., character + magic system = character with unique relationship to magic). User writes the hybrid concept.**                                            |
| **Expand This**          | **For an idea with minimal content, AI suggests questions to explore (e.g., "What does this character want? What's in their way? What would they never do?"). User answers in their own words.**                                   |


**Idea Evolution Tracking:**

- **View an idea's full history: from seed → sprout → notes → harvest**
- **Branching paths: if an idea splits into multiple directions (e.g., character evolves two ways), track each branch separately with parent/child relationships**
- **Visual timeline of development with milestones marked**
- **"Time capsule" view shows idea as it existed at any past date**

**Incubation Mode:**

- **For ideas that need time to develop subconsciously**
- **Scheduled reminders to revisit after set period (7 days, 30 days, custom)**
- **Gentle notifications: "Your idea about [title] has been incubating for 30 days. Any new thoughts?"**
- **Option to receive incubation prompts via email or in-app only**
- **Progress indicator shows incubation time remaining**

**Idea Garden Dashboard:**

- **Visual dashboard showing idea counts by status, recent activity, and "ready to harvest" suggestions**
- **"Bloom" indicator for ideas that have received consistent attention over time**
- **Seasonal themes (spring: new growth visuals; autumn: harvest suggestions)**
